## **Método de encriptación**
El programa reemplazara las vocales de la frase ingresada siguiendo, el valor de la tabla, para el proceso de desencriptación de forma inversa.

| Clave  | Valor  |
| ------------ | ------------ |
| a  | enter  |
| e | ines  |
| i  | ai  |
| o  | ober  |
| u  | ufat  |
